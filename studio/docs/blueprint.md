# **App Name**: DPI Navigator

## Core Features:

- DPI Question Answering: Answer questions related to Digital Public Infrastructure using provided documentation as a tool for LLM reasoning.
- DPI Suggestion: Suggest relevant DPIs based on country context using provided documentation as a tool for LLM reasoning.
- Result Display: Display search results clearly with links to source documentation.
- Chat Interface: Provide a simple chat interface for users to interact with the assistant.
- Data Processing: Ingest and process data from docs.cdpi.dev, attached markdown file and transcribe relevant YouTube videos for LLM consumption.
- Feedback Mechanism: Allow the users to provide feedback (e.g. thumbs up or thumbs down) on the quality of responses to help refine the model's responses.

## Style Guidelines:

- Primary color: Strong blue (#29ABE2) to represent trust, technology, and global connectivity.
- Background color: Light gray (#F5F5F5) to ensure readability and a clean interface.
- Accent color: Teal (#008080) for interactive elements, call to actions and highlighting important information.
- Body and headline font: 'Inter', a sans-serif font known for its legibility and modern appearance.
- Use clear, consistent icons from a standard library (e.g., Font Awesome) to represent different DPI topics.
- Maintain a clean and intuitive layout that prioritizes ease of use and information accessibility.
- Use subtle animations and transitions to enhance user experience without being distracting.
